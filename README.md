# CapFourmi

Capistrano 3 recipes

## Installation

Add this line to your application's Gemfile:

    gem 'cap_fourmi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cap_fourmi

## Usage

```
# Gemfile
group :development do
  gem 'cap_fourmi'
end
```

```
# Capfile
require "cap_fourmi/all"
```
